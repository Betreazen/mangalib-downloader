# MangaLib Downloader

Инструмент для скачивания глав манги с [MangaLib](https://mangalib.me/) через их
публичный JSON API. Десктоп-приложение на **PySide6**. Поддерживает **выбор
перевода** (несколько команд на один тайтл), сохранение **оригиналов (AVIF)** и
**конвертацию в JPG**, упаковку в **CBZ**, **токен авторизации** для лицензированных
тайтлов и **защиту от лимитов** сервера.

## Возможности

- Вставляешь ссылку на тайтл или главу → подгружается список глав.
- **Выбор перевода (ветки).** Идентичность перевода — это `branch_id`. Команды
  внутри одной ветки могут меняться от главы к главе и/или отсутствовать, поэтому:
  - в списке переводов показан охват: «N глав (гл. A–B)» и сводка команд;
  - в таблице глав у **каждой** главы видно, кто её перевёл (или «Без указания
    команды», если перевод не подписан).
- Переводы могут покрывать **не всю** мангу — у каждого свой диапазон глав, это
  учтено (для выбранной ветки показываются только её главы).
- Выбор глав галочками, диапазоном (`1-10, 15, 20`), «выделить/снять все».
- Сохранение: для каждой главы создаётся
  - `original/` — исходные файлы как на сайте (обычно AVIF),
  - `converted/` — конвертированные JPG (максимальная совместимость),
  - `… [original].cbz` и `… [converted].cbz`.
- **Защита от лимитов:** токен-бакет (ограничение запросов/сек), обработка
  `HTTP 429` с уважением `Retry-After`, ретраи с экспоненциальной паузой,
  переключение между CDN-серверами, настраиваемая пауза между главами. Скорость,
  параллельность и пауза регулируются в интерфейсе.
- **Браузер каталога в GUI.** Кнопка «Каталог…» — поиск по названию, фильтр по типу
  (манга/манхва/маньхуа), сортировка, подгрузка «ещё» (та самая постраничная выдача).
  Выбираешь тайтл → он сразу грузится.
- **Докачка (пропуск уже скачанных).** Опция «Пропускать уже скачанные» — при
  повторном запуске берутся только недостающие главы. Это же решает «забрать платную
  главу после `expired_at`»: перезапустил позже — подтянутся только освободившиеся.
- **Токен** сохраняется между запусками (по желанию) в домашней папке.
- **Платные / отложенные главы.** На MangaLib ранний доступ и отложенный релиз —
  это один механизм: глава заперта (`expired_type=1`, `restricted_view.is_open=false`)
  и авто-открывается в дату `expired_at` (или сразу за оплату). Такие главы видны в
  списке, но при скачивании помечаются как 🔒 платные (с ценой и датой открытия),
  пропускаются и считаются отдельно — не ломают загрузку остальных. Невышедшие
  (`publish_at` в будущем) помечаются как ⏰.
- **Устойчивость к «грязным» данным.** Данные MangaLib местами битые (элемент ветки
  или команды бывает строкой вместо объекта; деталь главы может отдавать 404) —
  парсинг это переживает и не падает.

## Каталог

Каталог на сайте — «бесконечный скролл», но под капотом это обычная постраничная
выдача API: `GET /api/manga?site_id[]=1&page=N` (по 60 на страницу). Признак конца —
`meta.has_next_page=false`; `meta.seed` нужно передавать на следующих страницах,
чтобы порядок не «прыгал». Метод `MangaLibClient.get_catalog(page, ...)` это
инкапсулирует (поддерживает `query`, `types`, `sort_by`).

## Установка

```bash
pip install -r requirements.txt
```

Требуется Python 3.11+. Поддержка AVIF — нативно через Pillow 12.

## Запуск (GUI)

```bash
python app.py
```

Вставь ссылку → «Загрузить» → выбери перевод и главы → укажи папку → «Скачать».

## Запуск (CLI)

```bash
python cli.py info "https://mangalib.me/ru/manga/7965--chainsaw-man"
python cli.py branches 7965--chainsaw-man
python cli.py download 7965--chainsaw-man --chapters 1-5 --branch 4666 -o downloads

# лицензированные / 18+ тайтлы — с токеном
python cli.py --token "ТОКЕН" download 247--shingeki-no-kyojin --chapters 1 --branch 1
```

## Bearer-токен (для лицензированных / 18+ тайтлов)

Большинству манги токен не нужен. Если список глав пустой — тайтл лицензирован
или 18+. Как достать токен:

1. Залогинься на mangalib.me.
2. **F12** → вкладка **Network** → в фильтре `api2` → обнови страницу.
3. Открой любой запрос к `api2.mangalib.me` → **Request Headers** →
   `Authorization: Bearer eyJ...` → скопируй значение после `Bearer `.

Вставь токен в поле в GUI (можно «Запомнить токен») или передай через `--token`.
Токен — личный секрет и со временем протухает; когда лицензированный тайтл
перестанет качаться, достань его заново тем же способом. Хранится локально в
`~/.mangalib_downloader/config.json`.

## Архитектура

Ядро (`mangalib_dl/`) не зависит от интерфейса — его используют и GUI, и CLI.

| Модуль | Назначение |
|---|---|
| `api.py` | клиент API: ссылки, метаданные, главы, ветки, страницы, серверы |
| `models.py` | модели: `Manga`, `Chapter`, `Branch`, `Page` |
| `ratelimit.py` | токен-бакет, разбор `Retry-After`, запрос с ретраями |
| `downloader.py` | параллельная загрузка страниц, детекция формата, AVIF→JPG |
| `packager.py` | сборка CBZ, безопасные имена файлов |
| `service.py` | оркестрация: выбранные главы выбранного перевода |
| `storage.py` | сохранение токена и настроек между запусками |
| `app.py` | десктоп-GUI на PySide6 (загрузка/скачка в фоновом `QThread`) |
| `cli.py` | командный интерфейс |

### Используемый API MangaLib

- `GET /api/manga/{slug}` — метаданные тайтла.
- `GET /api/manga/{slug}/chapters` — список глав; у каждой массив `branches`
  (ветки-переводы) с `branch_id` и командами.
- `GET /api/manga/{slug}/chapter?number=&volume=&branch_id=` — страницы главы.
- `GET /api/constants?fields[]=imageServers` — список CDN-серверов.

URL картинки = сервер + `page.url`, напр.
`https://img2.imglib.info` + `/manga/.../00.png_res.jpg`.
